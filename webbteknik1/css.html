Vad är en regel i CSS och vad består den av?

Den består av Selektor och Deklaration. Selektor besttämmer vilken element som ska påverkas t.ex. 'p' för alla stycken eller en specifik klass med klassnamn. deklaration. bestämmer vilken stil eller hur elementet ska se ut, t.ex. color: red för att ändra texten till röd.

Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Skillnaden mellan id och class är att id måste vara unikt för varje HTML-dokument, vilket betyder att bara ett element kan ha ett specifikt id. Id används endast en gång per sida. däremot class kan återanvändas för flera element, vilket är praktiskt när man vill applicera samma stil på flera delar av sidan. Id har också högre specifikitet än class, så om ett element har både id och class kommer stilen för id att prioriteras. Class kan kombineras med andra selektorer för att öka specifikiteten. Använd id för unika element och class för att styla flera likadant.
